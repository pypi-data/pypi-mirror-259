# C99 规范

## 下载

### Docker

```
docker pull apachecn0/c99-guifan
docker run -tid -p <port>:80 apachecn0/c99-guifan
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install c99-guifan
c99-guifan <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g cjiujiu-guifan
cjiujiu-guifan <port>
# 访问 http://localhost:{port} 查看文档
```
