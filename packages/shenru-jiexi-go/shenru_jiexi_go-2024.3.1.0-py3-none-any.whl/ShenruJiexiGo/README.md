# 深入解析Go

## 下载

### Docker

```
docker pull apachecn0/shenru-jiexi-go
docker run -tid -p <port>:80 apachecn0/shenru-jiexi-go
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install shenru-jiexi-go
shenru-jiexi-go <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g shenru-jiexi-go
shenru-jiexi-go <port>
# 访问 http://localhost:{port} 查看文档
```
