# 深入理解Yii2.0

## 下载

### Docker

```
docker pull apachecn0/shenru-lijie-yii2-0
docker run -tid -p <port>:80 apachecn0/shenru-lijie-yii2-0
# 访问 http://localhost:{port} 查看文档
```

```
pip install shenru-lijie-yii2-0
shenru-lijie-yii2-0 <port>
# 访问 http://localhost:{port} 查看文档
```

```
# 访问 http://localhost:{port} 查看文档
```
