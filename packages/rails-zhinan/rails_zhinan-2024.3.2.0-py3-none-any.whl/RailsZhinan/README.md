# Rails 指南

## 下载

### Docker

```
docker pull apachecn0/rails-zhinan
docker run -tid -p <port>:80 apachecn0/rails-zhinan
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install rails-zhinan
rails-zhinan <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g rails-zhinan
rails-zhinan <port>
# 访问 http://localhost:{port} 查看文档
```
