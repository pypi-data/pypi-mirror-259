# d3-api-ref

## 下载

### Docker

```
docker pull apachecn0/d3-api-ref
docker run -tid -p <port>:80 apachecn0/d3-api-ref
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install d3-api-ref
d3-api-ref <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g d3-api-ref
d3-api-ref <port>
# 访问 http://localhost:{port} 查看文档
```
