# Got Github

## 下载

### Docker

```
docker pull apachecn0/got-github
docker run -tid -p <port>:80 apachecn0/got-github
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install got-github
got-github <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g got-github
got-github <port>
# 访问 http://localhost:{port} 查看文档
```
