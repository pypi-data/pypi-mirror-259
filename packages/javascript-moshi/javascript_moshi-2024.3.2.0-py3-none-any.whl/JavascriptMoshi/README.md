# JavaScript 模式

## 下载

### Docker

```
docker pull apachecn0/javascript-moshi
docker run -tid -p <port>:80 apachecn0/javascript-moshi
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install javascript-moshi
javascript-moshi <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g javascript-moshi
javascript-moshi <port>
# 访问 http://localhost:{port} 查看文档
```
