# 你好，C++！

## 下载

### Docker

```
docker pull apachecn0/nihao-c
docker run -tid -p <port>:80 apachecn0/nihao-c
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install nihao-c
nihao-c <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g nihao-c
nihao-c <port>
# 访问 http://localhost:{port} 查看文档
```
