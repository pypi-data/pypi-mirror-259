# 一起学 koa

## 下载

### Docker

```
docker pull apachecn0/yiqi-xue-koa
docker run -tid -p <port>:80 apachecn0/yiqi-xue-koa
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install yiqi-xue-koa
yiqi-xue-koa <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g yiqi-xue-koa
yiqi-xue-koa <port>
# 访问 http://localhost:{port} 查看文档
```
