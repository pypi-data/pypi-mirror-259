# zeptojs-api-doc

## 下载

### Docker

```
docker pull apachecn0/zeptojs-api-doc
docker run -tid -p <port>:80 apachecn0/zeptojs-api-doc
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install zeptojs-api-doc
zeptojs-api-doc <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g zeptojs-api-doc
zeptojs-api-doc <port>
# 访问 http://localhost:{port} 查看文档
```
