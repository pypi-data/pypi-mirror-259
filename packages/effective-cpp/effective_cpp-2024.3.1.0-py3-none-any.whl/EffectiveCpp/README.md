# effective-cpp

## 下载

### Docker

```
docker pull apachecn0/effective-cpp
docker run -tid -p <port>:80 apachecn0/effective-cpp
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install effective-cpp
effective-cpp <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g effective-cpp
effective-cpp <port>
# 访问 http://localhost:{port} 查看文档
```
