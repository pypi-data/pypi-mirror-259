# 软件构建实践 0.1

## 下载

### Docker

```
docker pull apachecn0/ruanjian-goujian-shijian-0-1
docker run -tid -p <port>:80 apachecn0/ruanjian-goujian-shijian-0-1
# 访问 http://localhost:{port} 查看文档
```
