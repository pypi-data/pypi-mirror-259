# re-for-beginners

## 下载

### Docker

```
docker pull apachecn0/re-for-beginners
docker run -tid -p <port>:80 apachecn0/re-for-beginners
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install re-for-beginners
re-for-beginners <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g re-for-beginners
re-for-beginners <port>
# 访问 http://localhost:{port} 查看文档
```
