# Tornado 概览

## 下载

### Docker

```
docker pull apachecn0/tornado-gailan
docker run -tid -p <port>:80 apachecn0/tornado-gailan
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install tornado-gailan
tornado-gailan <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g tornado-gailan
tornado-gailan <port>
# 访问 http://localhost:{port} 查看文档
```
