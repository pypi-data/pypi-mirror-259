# SICP in Python

## 下载

### Docker

```
docker pull apachecn0/sicp-in-python
docker run -tid -p <port>:80 apachecn0/sicp-in-python
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install sicp-in-python
sicp-in-python <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g sicp-in-python
sicp-in-python <port>
# 访问 http://localhost:{port} 查看文档
```
