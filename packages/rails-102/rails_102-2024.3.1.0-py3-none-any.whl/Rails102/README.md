# Rails 102

## 下载

### Docker

```
docker pull apachecn0/rails-102
docker run -tid -p <port>:80 apachecn0/rails-102
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install rails-102
rails-102 <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g rails-yilinger
rails-yilinger <port>
# 访问 http://localhost:{port} 查看文档
```
