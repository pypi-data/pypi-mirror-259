# C4iOS - COSMOS

## 下载

### Docker

```
docker pull apachecn0/c4ios-cosmos
docker run -tid -p <port>:80 apachecn0/c4ios-cosmos
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install c4ios-cosmos
c4ios-cosmos <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g c4ios-cosmos
c4ios-cosmos <port>
# 访问 http://localhost:{port} 查看文档
```
