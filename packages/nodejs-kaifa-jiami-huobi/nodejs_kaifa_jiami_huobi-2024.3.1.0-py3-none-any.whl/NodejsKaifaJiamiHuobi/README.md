## 下载

```
docker pull apachecn0/nodejs-kaifa-jiami-huobi
docker run -tid -p <port>:80 apachecn0/nodejs-kaifa-jiami-huobi
# 访问 http://localhost:{port} 查看文档
```

```
pip install nodejs-kaifa-jiami-huobi
nodejs-kaifa-jiami-huobi <port>
```
