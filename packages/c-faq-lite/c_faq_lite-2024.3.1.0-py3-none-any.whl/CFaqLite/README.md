# C++ FAQ Lite

## 下载

### Docker

```
docker pull apachecn0/c-faq-lite
docker run -tid -p <port>:80 apachecn0/c-faq-lite
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install c-faq-lite
c-faq-lite <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g c-faq-lite
c-faq-lite <port>
# 访问 http://localhost:{port} 查看文档
```
