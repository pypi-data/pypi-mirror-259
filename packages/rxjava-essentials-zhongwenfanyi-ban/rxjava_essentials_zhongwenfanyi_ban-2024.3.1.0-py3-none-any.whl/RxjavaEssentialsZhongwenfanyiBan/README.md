# RxJava Essentials 中文翻译版

## 下载

### Docker

```
docker pull apachecn0/rxjava-essentials-zhongwenfanyi-ban
docker run -tid -p <port>:80 apachecn0/rxjava-essentials-zhongwenfanyi-ban
# 访问 http://localhost:{port} 查看文档
```
