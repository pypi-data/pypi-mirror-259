# 一起学 Node.js

## 下载

### Docker

```
docker pull apachecn0/yiqi-xue-node-js
docker run -tid -p <port>:80 apachecn0/yiqi-xue-node-js
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install yiqi-xue-node-js
yiqi-xue-node-js <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g yiqi-xue-node-js
yiqi-xue-node-js <port>
# 访问 http://localhost:{port} 查看文档
```
