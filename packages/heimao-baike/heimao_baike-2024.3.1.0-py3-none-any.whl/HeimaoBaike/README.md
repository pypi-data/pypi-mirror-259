# 黑帽百科

## 下载

### Docker

```
docker pull apachecn0/heimao-baike
docker run -tid -p <port>:80 apachecn0/heimao-baike
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install heimao-baike
heimao-baike <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g heimao-baike
heimao-baike <port>
# 访问 http://localhost:{port} 查看文档
```
