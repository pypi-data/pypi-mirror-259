# PHP安全基础详解

## 下载

### Docker

```
docker pull apachecn0/php-anquan-jichu-xiangjie
docker run -tid -p <port>:80 apachecn0/php-anquan-jichu-xiangjie
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install php-anquan-jichu-xiangjie
php-anquan-jichu-xiangjie <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g php-anquan-jichu-xiangjie
php-anquan-jichu-xiangjie <port>
# 访问 http://localhost:{port} 查看文档
```
