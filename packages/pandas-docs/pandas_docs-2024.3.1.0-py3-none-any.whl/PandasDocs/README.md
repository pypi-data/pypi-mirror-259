# pandas docs

## 下载

### Docker

```
docker pull apachecn0/pandas-docs
docker run -tid -p <port>:80 apachecn0/pandas-docs
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install pandas-docs
pandas-docs <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g pandas-docs
pandas-docs <port>
# 访问 http://localhost:{port} 查看文档
```
