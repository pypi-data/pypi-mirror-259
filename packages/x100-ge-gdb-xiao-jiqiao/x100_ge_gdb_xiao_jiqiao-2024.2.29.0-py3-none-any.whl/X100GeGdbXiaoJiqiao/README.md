# 100个gdb小技巧

```
docker pull apachecn0/x100-ge-gdb-xiao-jiqiao
docker run -tid -p <port>:80 apachecn0/x100-ge-gdb-xiao-jiqiao
# 访问 http://localhost:{port} 查看文档
```
